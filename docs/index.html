// system architecture

One platform, two planes

MatrixHub Cloud is the SaaS control plane — catalog, workspaces, jobs and policies. Matrix Runtime is the execution plane that runs inside your network. The model gateway routes every model call; the CLI drives it all.

// what the runtime does

Verified execution, by job type

Every unit of work is a typed, TTL-bounded job that streams lifecycle events. Test before you trust.

10-minute sandboxes

Spin up any MCP server over stdio, run initialize and tools/list, call tools live, then auto-expire. No production secrets.

mcp.teststdio

Jobs & live events

Server-Sent Events stream every step — validate → ready — with status, logs and results, cancellable mid-flight.
